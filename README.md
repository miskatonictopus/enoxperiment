## Tech Stack

This project is built with **Next.js 16** and **React 19**, using **TypeScript** for type safety and maintainability.

### Core technologies
- **Next.js 16** (App Router) for the application framework
- **React 19** + **React DOM** for UI rendering
- **TypeScript** for static typing
- **Tailwind CSS v4** for styling and responsive layouts
- **shadcn/ui** + **Base UI** for reusable UI primitives
- **Lucide React** for iconography

### Creative rendering / visuals
- **p5.js** for generative sketches and shader-driven visual experiments
- **Three.js** with **@react-three/fiber** and **@react-three/drei** for 3D/WebGL scenes
- **postprocessing** for visual effects pipelines

### Utility and UX helpers
- **html2canvas** for client-side snapshot/capture generation
- **next-themes** for theme management
- **clsx**, **class-variance-authority**, and **tailwind-merge** for class composition patterns

### Tooling
- **ESLint** + `eslint-config-next` for linting
- **Prettier** + `prettier-plugin-tailwindcss` for formatting
- **PostCSS** + **Tailwind CSS** build pipeline
